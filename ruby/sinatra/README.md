# hello-world-ruby-sinatra

Ruby Sinatra based hello world ready to run.  

1. Clone repo  
```git clone https://github.com/jbpadgett/hello-world-repo.git```  

2. Cd into ruby rails hello world dir  
```cd ruby/sinatra```  

3. Execute the app using Sinatra
Rack
```rackup config.ru```

Directly
```ruby hello.rb```
